# 🌅 Sur les pas d'Aurore - Site Web Officiel

Bienvenue sur le dépôt officiel du groupe **Sur les pas d'Aurore**.
Ce site est développé avec **Next.js 14**, **Tailwind CSS**, hébergé sur **Vercel** et administré depuis **Google Drive & Google Sheets**.

## 🛠️ Architecture 100% Cloud (Sans installation locale)
1. **Google Drive & Sheets** : Vous gérez les textes, dates de concert, membres et photos.
2. **GitHub Web** : Héberge le code source. Toute mise à jour de fichier se fait en ligne.
3. **Vercel** : Publie le site instantanément.
